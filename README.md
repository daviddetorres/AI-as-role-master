# AI-as-role-master
Prompts that allows you to use any LLM as a master for your role playing games. 

# Instructions
1. Choose an LLM. Some options are: 
    - DeepSeek AI (this is the one that I use the most, as it's teh best keeping the coherence of the story)
    - OpenAI GPT-4
    - Anthropic Claude
    - Google Gemini

2. Find a prompt for the rules and style of the RPG you want to play from the section "Rules prompts" below.
3. Copy it and paste it in your LLM of choice.
4. You will be asked to define your story, ambience, characters, etc. Answer the questions as best as you can.
5. Start playing!


# Rules prompts
In the file [rules-prompts.md](rules-prompts.md) you will find a collection of prompts for different RPG systems and styles. Choose the one that fits your needs the best!
